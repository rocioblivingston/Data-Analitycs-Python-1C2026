# 📊 Data Analytics: Marketing, Clientes & Ventas 🚀
### Proyecto Trabajo de Finalización de curso - 1C 2026

## 📝 Descripción del Proyecto
Este repositorio contiene el desarrollo del **Análisis Exploratorio de Datos (EDA)** enfocado en el ecosistema comercial. Se analizan datasets de **ventas, comportamiento de clientes y métricas de marketing** utilizando Python para transformar datos crudos en insights estratégicos.

## 📁 Estructura del Repositorio
* 📂 `data/`: Datasets con información de transacciones, perfiles de clientes y campañas.
* 📓 `notebooks/`: Jupyter Notebooks con el desarrollo del análisis paso a paso.
* 📜 `scripts/`: Archivos de Python para limpieza de datos y funciones auxiliares.
* 🖼️ `visualizations/`: Gráficos clave generados durante el proceso.

## 🛠️ Tecnologías y Librerías
* 🐍 **Python:** Lenguaje base para el procesamiento de datos.
* ![Pandas](https://img.shields.io/badge/pandas-%23150458.svg?style=flat&logo=pandas&logoColor=white) **Pandas**
* ![NumPy](https://img.shields.io/badge/numpy-%23013243.svg?style=flat&logo=numpy&logoColor=white) **NumPy**
* 📈 **Matplotlib & Seaborn:** Visualización de datos.

## ⚙️ Proceso de Análisis
1.  **Exploración de Ventas:** 💰 Identificación de tendencias y productos estrella.
2.  **Análisis de Clientes:** 👥 Estudio demográfico y patrones de compra.
3.  **Rendimiento de Marketing:** 📣 Evaluación de canales y efectividad de conversión.
4.  **Data Wrangling:** 🧹 Limpieza y normalización de la base de datos.
